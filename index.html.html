歸零，是為了更好的開始

以前，我是教學醫院的資深護理師，日子在輪班與高壓中度過。生活被填得很滿，卻常覺得身心透不過氣。

所以，我做了一個決定 —— 讓人生「歸零」，像這抹雲舞白一樣，為自己留出空間。

我加入了 雅筑國際 Easecox Group。
這不是衝動，而是我看見了未來的趨勢。

健康是剛需，美是天性

雅筑的產品結合了我在醫學領域重視的「健康機能」與女性渴望的「體態美」。推廣 Diamond 寶石衣，其實是在幫助人用最簡單的方式找回健康。

像雲一樣的自由流動

創業後，我不再被打卡鐘綁架。我有更多的時間陪伴家人、照顧自己。這種「時間的留白」，正是 Cloud Dancer 想要傳遞的生活哲學。

重新定義自我的畫布

在這裡，妳不需要很厲害才能開始，但妳要開始了才會很厲害。雅筑提供完整的培訓系統，就像在白紙上構建新故事。

這張白紙已經準備好了

Cloud Dancer 告訴我們：放下沈重的負擔，簡單即是最好。
如果妳也想擁有一份能兼顧健康、財富與自由的事業，
我願意以過來人的經驗，成為妳的引路人。
